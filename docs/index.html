Welcome!

This documentation provides a guide to using the GeoArchaeology Terrain Processor application. This powerful tool is designed to help archaeologists and researchers analyze terrain and satellite imagery to discover, investigate, and document archaeological sites and landscapes.

Whether you are working with your own LiDAR data (LAZ/LAS files) or defining areas of interest directly on the map, this application offers a suite of tools to generate insightful raster visualizations and perform vegetation analysis.

Main Use Cases

The application supports several key workflows for archaeological investigation:

1. Loading Your Own LiDAR Data (LAZ/LAS Files)

Import individual LAZ/LAS files or entire folders of LiDAR data. The system automatically processes this data, determines its geographic extent, and prepares it for further analysis. You can also enable NDVI processing to acquire corresponding satellite imagery.

Learn more about Loading LAZ/LAS Files...

2. Defining Regions of Interest from the Map

Interactively define study areas by clicking a single point, drawing a path (for transects or linear features), or drawing a box (for broader areas). You can specify how many sub-regions to create from paths/boxes and enable NDVI processing.

Learn more about Creating Regions from the Map...

3. Retrieving Elevation and Satellite Data

For regions defined on the map, use the "Get Data" function to fetch relevant elevation data (e.g., DTM from USGS 3DEP) and Sentinel-2 satellite imagery (if NDVI is enabled). This automatically triggers the generation of various terrain rasters.

Learn more about the "Get Data" Function...

4. NDVI Analysis for Vegetation Insights

Leverage Sentinel-2 satellite imagery to calculate and visualize the Normalized Difference Vegetation Index (NDVI). This can help identify subtle vegetation patterns related to buried archaeological features or past land use.

Learn more about NDVI Analysis...

5. Analyzing Generated Raster Visualizations

Explore a rich set of automatically generated raster products like Digital Terrain Models (DTM), Hillshades, Slope maps, Aspect maps, Color Relief, and more. These visualizations are crucial for identifying and interpreting archaeological features.

Learn more about Understanding Rasters...

Getting Started

To begin, you can either:

  • Go to the Loading LAZ/LAS Files section if you have your own LiDAR data.
  • Or, learn how to Create Regions from the Map if you want to define new areas of interest.

For understanding the various data products you'll be working with, refer to the Raster Guide.
